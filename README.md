# Personal Projects

This repository contains a collection of personal and academic projects developed in different technical areas, including embedded systems, web development, CAD design, and power electronics.

## Projects

### Embedded programming: Washing machine application
- making use of an Atmega32 microcontroller set on the EasyAVR v7 board
- I designed an application for a 2-mode washing machine using timers and PWM signals

### AutoCAD and AutoCAD Plant
- using AutoCAD I designed a mechanical piece
- using AutoCAD Plant I designed the hydraulic scheme of an installation

### Power electronics: Design and simulation of a Buck converter
- step by step design and calculations for a Buck converter
- includes real component suggestions and simulated responses

### Database Frontend: Volleyball gear online store
- developed a web application using HTML, PHP, and SQL for an online volleyball gear store
- implemented user authentication, product catalog browsing, basket and wishlist features, product reviews, and order management
